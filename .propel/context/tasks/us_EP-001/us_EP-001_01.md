# User Story: Admin Dashboard

## Epic: EP-011

### As an admin,
I want to access a comprehensive dashboard after login,
so that I can efficiently manage users, roles, audit logs, and system settings.

#### Acceptance Criteria
- Admins are directed to the admin dashboard after login.
- The dashboard displays user management options (create, update, deactivate, assign roles).
- Audit logs and user activity are accessible and filterable.
- System settings and configuration options are available.
- Security alerts and compliance status are visible.
- All actions are logged immutably for audit.

#### Business Value
Centralizes administrative controls, improves compliance, and streamlines user management.

#### Effort Estimation
5 Story Points

# User Story: Patient Dashboard / Landing Screen

## Epic: EP-010

### As a patient,
I want to access a dashboard or landing screen after login,
so that I can easily view and manage all aspects of my healthcare in one place.

#### Acceptance Criteria
- Upon login, patients are directed to a dashboard/landing screen.
- The dashboard displays upcoming and past appointments.
- Patients can book, reschedule, or cancel appointments from the dashboard.
- Patients can upload clinical documents and view uploaded files.
- Intake forms are accessible and can be completed from the dashboard.
- Notifications and reminders are visible and actionable.
- Insurance pre-check status and appointment confirmations (PDF) are accessible.
- Medication conflict alerts and medical coding (ICD-10, CPT) info are visible if applicable.
- The dashboard is secure and HIPAA-compliant.

#### Business Value
Empowers patients, increases engagement, and streamlines access to all key features.

#### Effort Estimation
8 Story Points

# User Story: Patient Appointment Booking

## Epic: EP-001

so that I can easily schedule my healthcare visits at my convenience.

# User Story: Automated Reminders for Appointments

## Epic: EP-001, EP-008

### As a patient,
I want to receive automated reminders and calendar sync options for my appointments,
so that I am less likely to forget or miss my scheduled visits.

#### Acceptance Criteria
- The system sends reminders via SMS/Email before each appointment.
- Patients can add appointments to their Google/Outlook calendar.
- Reminders are sent at configurable intervals (e.g., 24h, 1h before).
- Staff can view reminder delivery status.

#### Business Value
Reduces no-shows, improves patient engagement.

#### Effort Estimation
5 Story Points
