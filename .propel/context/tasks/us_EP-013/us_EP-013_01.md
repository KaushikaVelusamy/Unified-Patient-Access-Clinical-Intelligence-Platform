# User Story: Restrict Patient Self-Check-In

## Epic: EP-013

### As a patient,
I must not be able to self-check-in via app, web, or QR code,
so that process control and compliance are maintained by staff.

#### Acceptance Criteria
- The system blocks all patient self-check-in attempts.
- Patients are shown a clear message explaining the restriction.
- All self-check-in attempts are logged for audit.
- Only staff can mark arrivals and manage check-ins.

#### Business Value
Ensures process control, supports compliance.

#### Effort Estimation
3 Story Points
