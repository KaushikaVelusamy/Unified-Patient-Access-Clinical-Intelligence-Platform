# User Story: Patient Intake (AI/Manual)

## Epic: EP-002

### As a patient,
I want to choose between AI-assisted conversational intake and manual forms,
so that I can complete my intake in the way that is most comfortable for me.

#### Acceptance Criteria
- The system allows patients to select AI or manual intake at any time.
- Intake forms are user-friendly and accessible.
- AI-assisted intake provides guided, conversational data entry.
- Patients can switch between AI/manual modes mid-process.
- Intake data is stored and available for staff review.
- Confirmation is provided upon successful intake submission.

#### Business Value
Flexible, efficient data collection, supports all patient types.

#### Effort Estimation
8 Story Points
